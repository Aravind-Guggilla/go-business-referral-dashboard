# Go Business Referral Dashboard

A responsive referral management dashboard built with React and Vite.
Live Link https://go-business-referral-dashboard-three.vercel.app

## Features

* User Authentication (JWT Cookie Based)
* Protected Routes
* Referral Dashboard
* Overview Metrics
* Service Summary
* Referral Link & Code Sharing
* Search Referrals
* Sort Referrals by Date
* Client-Side Pagination
* Referral Details Page
* 404 Not Found Page

## Tech Stack

* React
* Vite
* React Router DOM
* JS Cookie
* CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed on Vercel.
